# Phone Class (OOP)

Учебный проект по ООП на Python.

## 📌 Описание

Реализован класс Phone, который моделирует телефон.

### Атрибуты:
- brand — бренд
- model — модель
- price — цена
- color — цвет
- storage_gb — объем памяти
- is_in_stock — наличие на складе

### Методы:
- get_full_name() → возвращает "Бренд Модель"
- apply_discount(percent) → применяет скидку к цене
- check_availability() → сообщает о наличии
- __str__() → красивый вывод объекта

## ▶️ Запуск

```bash
python phone.py
